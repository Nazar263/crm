# Plan: Add Animations to CRM

## Goal
Add polished, premium animations throughout the CRM to make it feel alive and modern — matching the existing dark premium aesthetic.

## Files to Modify
- `style.css` — CSS keyframes, transitions, animation classes
- `js/ui.js` — Staggered entrance logic, counter animations, page transition triggers

## Animations to Add

### 1. Staggered Card Entrance (stat cards, chart cards, developer cards)
- Each card fades in + slides up with a 50ms stagger delay
- CSS class `.anim-stagger` with `animation: cardIn 0.4s ease both`
- JS applies `animation-delay` via inline style based on index

### 2. Number Counter Animation (stat values)
- When dashboard loads, stat values count up from 0 to target
- JS utility `animateCounter(el, target, duration)` with `requestAnimationFrame`
- Easing: ease-out for natural feel

### 3. Enhanced Page Transitions
- Replace simple `fadeIn` with a more dynamic transition
- Page content slides up + fades in, with sections staggering

### 4. Table Row Entrance
- Table rows slide in from left with opacity fade
- CSS `.data-table tbody tr` gets `animation: rowIn 0.3s ease both`
- Staggered via JS when rendering tables

### 5. Modal Spring/Bounce
- Replace `slideUp` with a spring-like bounce on open
- Close animation: scale down + fade out

### 6. Button Press Effect
- `:active` state scales down to 0.96 with a quick transition
- Adds tactile feel

### 7. Card Hover Glow
- Stat cards get a subtle colored glow on hover (matching their icon color)
- Enhanced `transform: translateY(-3px)` with glow shadow

### 8. Toast Spring Animation
- Replace `toastIn` with a spring overshoot (slides past final position then settles)

### 9. Progress Bar Fill Animation
- `.progress-bar__fill` animates width from 0 with a delay

### 10. Sidebar Nav Pulse on Active
- Active nav item gets a subtle breathing glow animation

## Implementation Order
1. CSS keyframes and utility classes (style.css)
2. Stagger logic + counter animation (js/ui.js)
3. Modal close animation (js/ui.js + style.css)
4. Table row stagger (js/ui.js — in renderTable calls)

## Verification
- Open index.html in browser
- Check dashboard: cards stagger in, numbers count up
- Switch pages: smooth transitions
- Open modals: bounce in, scale out on close
- Hover cards/buttons: glow and press effects
- Tables: rows slide in
